## ğŸ›  Installation

To compile and run programs:

1. Install GCC:
   ```bash
   # Ubuntu/Debian
   sudo apt-get install build-essential
   
   # CentOS/RHEL
   sudo yum install gcc
   
   # macOS
   xcode-select --install
   ```

2. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/oaip.git
   cd oaip
   ```

3. Compile the program:
   ```bash
   cd src/1_Sem/lab1
   gcc task1.c -o task1
   ```

4. Run the program:
   ```bash
   ./task1
   ```
